Traversing allow you to traverse through HTML node.

## Methods

| Name | Description |
| :--- | :---------- |
| SelectNodes() | Selects a list of nodes matching the XPath expression. |
| SelectSingleNode(String) | Selects the first XmlNode that matches the XPath expression. |
